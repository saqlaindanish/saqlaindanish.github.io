
# 🚀 Mostafa Hassan Portfolio

Modern and fully responsive personal portfolio website for a Junior Full Stack .NET Developer, built with HTML, CSS, JavaScript, and Tailwind CSS.

Designed with a premium modern UI, smooth animations, interactive effects, and Dark/Light mode support to showcase skills, experience, and developer identity professionally.

---

## ✨ Features

* 🎨 Modern & Clean UI Design
* 🌙 Dark / Light Mode
* ⚡ Smooth Animations & Transitions
* 📱 Fully Responsive on All Devices
* 🖱 Interactive Hover Effects
* 🚀 Optimized Performance
* 🎯 Smooth Scrolling Navigation
* 💻 Developer-Focused Layout

---

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Tailwind CSS (CDN)
